# **App Name**: AdzPro Digital

## Core Features:

- Dynamic Page Rendering: Render core sections including Hero, About Us, and Services with hardcoded content and dynamically generated elements for consistency and ease of updates.
- Portfolio Showcase Module: Display high-end project case studies with rich media and detailed descriptions, inspired by the provided template style.
- Lead Capture Contact Form: Provide a functional contact form allowing prospective clients to submit inquiries directly.
- Firestore Contact Data Storage: Securely store all contact form submissions within a Firebase Firestore database for lead management.
- Automated Email Notifications: Utilize Firebase Cloud Functions to send an automated email notification to 'operation@adzpro.co.in' upon each new form submission.
- SEO Metadata Management: Implement crucial SEO meta tags for keywords such as 'Adz Pro New Delhi,' 'Advertising Agency in Bijwasan,' and 'Digital Marketing Expert Adz Pro' to enhance search engine visibility.
- Responsive & Modern User Interface: Ensure the website maintains a sleek, professional, and fully responsive design across all devices (mobile, tablet, desktop), incorporating high-quality transitions and glassmorphism effects.

## Style Guidelines:

- Primary Color: Electric Blue (#007BFF). A vibrant and energetic hue used for interactive elements, call-to-actions, and key highlights, conveying dynamism and professionalism.
- Background Color: Deep Navy (#1F262E). A rich, dark blue-gray base for the main layout, providing a sophisticated, 'tech-forward' canvas that makes content and brighter colors pop.
- Accent Color: Luminous Cyan (#44EDDA). An analogous, bright, and highly saturated cyan used sparingly for subtle modern touches, unique accents, and attention-grabbing elements, complementing the blues with a technological edge.
- Text Color: White (#FFFFFF). Employed for primary text and headings to ensure maximum readability against the dark backgrounds and blues, maintaining a clean and stark contrast.
- Headline and Body Font: 'Inter' (sans-serif). A modern, clean, and highly legible typeface selected for both headings and body text, supporting a 'Tech-Forward' aesthetic and ensuring excellent readability across all devices.
- Use sleek, minimalist, and professionally designed icons that complement the modern and tech-forward design aesthetic of AdzPro Digital, emphasizing clarity and functionality.
- Implement a clean, spacious, and grid-based layout for sections like 'Services' and 'Portfolio,' featuring high-impact hero sections and incorporating subtle glassmorphism effects on select UI components for depth and modernity.
- Incorporate smooth, high-quality transitions and subtle hover effects throughout the site to enhance user engagement and deliver a premium, responsive experience, aligning with a professional 'tech-forward' brand.
